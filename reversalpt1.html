שתי גישות לפתרון
בונים את הפתרון אלגברית מהגדרות MB ו-MS. לחצו על כל שלב לחשיפתו.

בסיס הכסף כולל מזומן בידי הציבור ורזרבה בנקאית:

MB = מב"צ + רזרבה בנקאית

כמות הכסף כוללת מזומן ופיקדונות עו"ש:

MS = מב"צ + עו"ש

MB כולל רזרבה בנקאית. MS כולל עו"ש. אלה שני דברים שונים לחלוטין.

מהגדרת rco: הציבור מחזיק rco יחידות מב"צ לכל יחידת עו"ש, לכן:

מב"צ = rco × עו"ש = 0.2 × עו"ש

מהגדרת r: הבנק שומר r יחידות רזרבה לכל יחידת עו"ש, לכן:

רזרבה בנקאית = r × עו"ש = 0.2 × עו"ש

מבטאים הכל כפונקציה של עו"ש, כדי שנוכל להציב בנוסחת MB ולחלץ ערך יחיד.

מציבים בנוסחת MB = מב"צ + רזרבה בנקאית:

MB = rco × עו"ש + r × עו"ש = (rco + r) × עו"ש

מחלקים את שני האגפים ב-(rco + r), מקבלים:

עו"ש = MBrco + r = 4000.2 + 0.2 = 4000.4 = 1,000

משתמשים בקשר מצעד 2:

מב"צ = rco × עו"ש = 0.2 × 1,000 = 200
בדיקה: MB = מב"צ + רזרבה = 200 + (0.2 × 1,000) = 200 + 200 = 400 ✓

מציבים מב"צ ועו"ש בהגדרת MS מצעד 1:

ΔMS = מב"צ + עו"ש = 200 + 1,000 = 1,200
💡 מה קרה לאותם 400?

200 הפכו למב"צ בידי הציבור. 200 הפכו לרזרבה בנקאית. הרזרבה של 200 תומכת ב-1,000 עו"ש (כי 200 = 0.2 × 1,000). סה"כ MS = 200 + 1,000 = 1,200, פי 3 מהעירוי.

נוסחה מקוצרת שנגזרת ישירות מהגדרות MB ו-MS. לחצו על כל שלב לחשיפתו.

מציבים rco = 0.2 ו-r = 0.2:

מכפיל = 1 + 0.20.2 + 0.2 = 1.20.4 = 3

המונה (1 + rco): על כל יחידת עו"ש יש גם rco יחידות מב"צ, ביחד (1+rco) יחידות MS.

המכנה (rco + r): שתי ה"דליפות" מהשרשרת, מה שהולך למזומן (rco) ומה שנשמר כרזרבה (r).

ΔMS = 400 × 3 = 1,200
💡 מתי המכפיל הוא 1/r = 5 ומתי 3?

המכפיל הבסיסי 1r = 5 מתקבל כאשר כמות המזומן בידי הציבור קבועה, כלומר כל עירוי נוסף מופקד במלואו בבנקים.

בשאלה שלנו הציבור מחזיק יחס קבוע rco = 0.2, לכן כל עירוי מחלחל גם למב"צ. הוספת rco למכנה מורידה את המכפיל: 1.20.4 = 3 במקום 5.
